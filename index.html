my portfolio

Jubilee Austen Project

My first project was creating a site for ficticious new developer Jubilee Austen. For an online tech course I built a one page static site from a Photoshop design comp.

Tools Used: Adobe Photoshop | HTML5 | CSS | Visual Studio Code | Google Fonts | W3C HTML & CSS Validation.

Highlight: Learning new technologies (VSC) and skills (floating)! It gave me a sense of wonder (like a child jumping in a puddle for the first time). In the end the clean and organized code filled me with satisfaction and sparked further curiosity of tech.

Challenge: Learning the tools and technologies above took quite a while. Each time I "jumped in a puddle" I wanted to stay there for hours and just learn.

Rogue Pickings Website

Tasked with translating a Photoshop design comp for a food truck business into HTML & CSS I focused on honing my boarders and floating skills.

Tools Used: Adobe Photoshop | HTML5 | CSS | Visual Studio Code | Google Fonts | W3C HTML & CSS Validation.

Highlight: New learning is always exciting for me - the box model specifically this time around.

Challenge: Focusing on borders and floats was challenging in that I had to deepen my understanding of the box model and how each one interacts. This required lots of Googling and article reading.

I had a lot of fun using the Jubilee Austen project as a template to create my first personal site.

Tools Used: Bluehost | Adobe Photoshop | HTML5 | CSS | Visual Studio Code | W3C HTML & CSS Validation.
